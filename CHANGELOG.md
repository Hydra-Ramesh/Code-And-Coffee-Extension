# Changelog

All notable changes to the **Code & Coffee Theme** extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [1.0.0] - 2024-01-01

### Added

- ☕ **Code & Coffee Dark** — Primary dark theme with warm espresso tones
- 🌙 **Code & Coffee Midnight** — Ultra-dark variant for late-night coding
- ☀️ **Code & Coffee Latte** — Soft light theme with creamy warmth
- 🎯 **Toggle Focus Mode** — Hide distractions for deep concentration
- ⏱️ **Start Deep Work Session** — 25-minute Pomodoro focus timer
- 🔔 **Coffee Break Reminder** — Hourly reminders to stretch and recharge
- Complete syntax highlighting for 15+ languages
- Full workbench customization (40+ UI surfaces)
- WCAG AA contrast compliance across all themes
- Professional marketplace documentation
