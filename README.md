## 🎨 Three Carefully Crafted Variants

### ☕ Code & Coffee Dark
> *Primary dark theme — warm espresso-brown editor with rich syntax colors.*

The signature experience. Deep brown backgrounds with caramel accents create a cozy atmosphere that's easy on the eyes for hours of coding.

### 🌙 Code & Coffee Midnight
> *Ultra-dark variant — near-black coffee-bean tones for late-night sessions.*

When the lights are off and the coffee is strong. Maximum contrast reduction for the darkest coding environments.

### ☀️ Code & Coffee Latte
> *Soft light theme — cream and warm brown tones for daytime coding.*

A warm alternative to harsh white themes. Latte-inspired backgrounds with rich brown text that feels natural and comfortable.

## ✨ Features

### 🎨 Complete Theme Coverage
- **40+ workbench surfaces** — every panel, sidebar, tab, and widget is themed
- **35+ token color rules** — comprehensive syntax highlighting
- **15+ languages** — JavaScript, TypeScript, React, Next.js, Node.js, Python, Go, Rust, Java, C#, HTML, CSS, JSON, YAML, Markdown
- **Semantic highlighting** — enhanced accuracy for supported languages

### ♿ Accessibility First
- **WCAG AA compliant** contrast ratios (4.5:1 minimum)
- Designed for **8+ hour coding sessions**
- **No neon or oversaturated colors** — gentle on the eyes
- **Color-blind friendly** — distinguishable without relying on color alone

### 🛠️ Productivity Commands

| Command | Description |
|---------|-------------|
| `Code & Coffee: Toggle Focus Mode` | Hide minimap, breadcrumbs, activity bar, and status bar for distraction-free coding |
| `Code & Coffee: Start Deep Work Session` | 25-minute Pomodoro timer with status bar countdown and progress notifications |
| `Code & Coffee: Coffee Break Reminder` | Hourly reminders with friendly coffee-themed messages |

Access all commands via the **Command Palette** (`Ctrl+Shift+P` / `Cmd+Shift+P`).

## 📦 Installation

### From VS Code Marketplace (Recommended)
1. Open **VS Code**
2. Go to **Extensions** (`Ctrl+Shift+X` / `Cmd+Shift+X`)
3. Search for **"Code & Coffee Theme"**
4. Click **Install**
5. Open **Command Palette** → **Preferences: Color Theme**
6. Select one of:
   - `Code & Coffee Dark`
   - `Code & Coffee Midnight`
   - `Code & Coffee Latte`

### From VSIX File
1. Download the `.vsix` file from [Releases](https://github.com/Hydra-Ramesh/Code-And-Coffee-Extension/releases)
2. Open VS Code
3. Run `Extensions: Install from VSIX...` from the Command Palette
4. Select the downloaded file

### From Command Line
```bash
code --install-extension code-and-coffee.code-and-coffee-theme
```
